Lucene App Crawler
==================

This repo is a CLI crawler for my dream [Lucene App](https://github.com/clitetailor/lucene-search-app).

Prerequisite
------------

> - Node (>= v8)

Installation
------------

To install run the following command from command prompt:

```bash
npm install
```

Usage
-----

```bash
node ./index.js --help
```
